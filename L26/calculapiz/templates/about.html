Calculapiz v0.0.1

Description

This is a service that performs calculations on demand. It is designed for practicing test automation skills.

Features

General terms

  • Service returns result as float value rounded to 3 digits after point.
  • Service doesn't have specific error handling. It is expected service responds with status code 400 to any error. All caught errors will be sent in response field error_details

Endpoints

Operations are available by endpoints like /api/v1/<operation>
v0.1 implements the following operations:
  • add(op1: float, op2:float) - returns sum of two operands
  • sub(op1: float, op2:float) - returns difference of two operands
  • mlp(op1: float, op2:float) - returns product of two operands
  • div(op1: float, op2:float) - returns quotient of two operands

Statistics

Every request is logged at /app/data/statistics.csv file. Successfully performed requests are logged as timestamp, status, operation, operand1, operand2, result. Failed requests are logged as timestamp, status, exception message.
